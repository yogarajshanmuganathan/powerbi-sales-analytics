**Description**
**End-to-end sales analytics dashboard built using Power BI. The project covers data modeling, DAX measures, time intelligence, and interactive visualizations.**

**Key Features**
KPI cards: Total Sales, Orders, Quantity
Time-series analysis using Date table
Category & region-level performance
Clean star-schema modeling
Optimized DAX measures

**Tools Used**
Power BI Desktop
SQL / PostgreSQL
DAX
Data Modeling
